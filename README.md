# Rust 学习笔记

[![License](https://img.shields.io/npm/l/heroicons.svg)](https://github.com/xuesongbj/Rust-Notes/blob/main/LICENSE)


Rust是一门系统级编程语言，和C、C++、Go等语言的设计思想有较大差异。学习Rust并不仅仅学习一门语言，更重要是学习它的设计思想。
