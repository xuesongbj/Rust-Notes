# Option

`Option` 是一个可选值，每个 `Option` 要么是 `Some` 并且包含一个值，要么 `None` 并且不包含。`Option`在Rust中很常见，因为它们有以下用途：

* 初始值
* 函数返回值
* 函数返回值包含错误信息的处理(错误时返回`None`)
* 可选的结构体字段
* 空指针
* ...
