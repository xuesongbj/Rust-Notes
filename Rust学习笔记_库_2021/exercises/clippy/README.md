# clippy

`clippy` 工具是一些列 `lint` 的集合，用于捕获常见错误和改进 `Rust` 代码。

如果要使用 `clippy` 工具，则应该安装 `Clippy`。如果没有，则需要通过手动安装它:

```bash
$> rustup component add clippy
```

对任何 Cargo项目运行 `clippy`的lint:

```bash
$> cargo clippy
```
