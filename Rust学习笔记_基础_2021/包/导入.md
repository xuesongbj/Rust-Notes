# 导入

我们可以在模块中以嵌套的方式导入(import)元素，这有助于减少导入操作的资源占用。

```rust
use std::sync::{Mutex, Arc, mpsc::channel};
use std::thread;

fn main() {
    let (tx, rx) = channel();

    let join_handle = thread::spawn(move || {
        while let Ok(n) = rx.recv() {
            println!("Received {}", n);
        }
    });

    for i in 0..10 {
        tx.send(i).unwrap();
    }

    join_handle.join().unwrap();
}
```
